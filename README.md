<h1>About</h1>

I made this project to highlight the absurd and violent history of sexuality detection and discrimination. Using a dataset I made myself and the Teachable Machine API, I created a webpage that tries to detect the user's sexuality based on physical attributes. Using a playful design, I hope to critique the way history has tried to associate the LGBTQ+ community with negative and create a false correlation betweensexual orientation and physical attributes.
